Contexte du projet

Ce projet consistait à développer un outil complet pour extraire les informations produits (nom, prix, caractéristiques) du site Darty et les stocker dans une base de données MySQL pour analyse ultérieure.

Les défis techniques principaux :

  • Contournement des protections anti-scraping du site.
  • Gestion des variations de structure entre pages produits.
  • Extraction robuste et fiable des données cibles.

L'architecture technique comprenait :

  • Un module d'extraction avec BeautifulSoup et Selenium.
  • Un ETL de transformation en Python.
  • Un système de stockage MySQL.

Résultats et Preuves

L'outil final permettait :

  • L'extraction de 1000+ produits/heure avec un taux de succès >95%.
  • Le stockage structuré dans une base MySQL avec historique des prix.
